This program can provide you below operations interactively and it does not ask any credentials to be included in code or during the operations. Please make sure AWS-CLI is configured properly.

Operations:
Create/Delete/List VPC.
Create/Delete/List  subnet in selected VPC.
Create/Delete/List internet gateway and attach it to the selected VPC.
Create/Delete/List route table in current VPC.
Associate/Deassociate subnet in/from a route table.
Create/Delete/List security groups in selected VPC.
Create/Delete/List load balancer.
Create/Delete/List target group with defined listener.
Register/Deregister instances into target group.
Create/Delete/List launch configuration.
Create/Delete/List auto scaling group.
Launch/Stop/Terminate an EC2 instance.
etc.
